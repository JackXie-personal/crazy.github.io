使用疯狂袋鼠AI的5分钟快速指南

返回控制台

注册与登录

通过首⻚「控制台」按钮或右上角【登录/注册】入口进入账号系统。支持以下方式：

  • 新用户：邮箱验证注册
  • 已有账号：用户名+密码登录
  • 第三方登录：支持GitHub账户授权

兑换使用额度

导航到「钱包」模块，按提示输入兑换码获取初始额度。

注意

  • 初始赠送 $0.01 (5000字符)
  • $1 = 500,000 字符
  • 充值后可申请发票

创建API密钥

  1. 进入「API令牌」模块
  2. 点击【创建新密钥】
  3. 配置参数：
    • 密钥名称（自由命名）
    • 有效期（最长1年）
    • 字符额度（不可超账户总额）
API端点：https://aiagent.crazykangaroo.space/v1

接口调用示例


# cURL 示例
curl https://aiagent.crazykangaroo.space/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <您的API密钥>" \
  -d '{
    "model": "gpt-4",
    "messages": [{"role": "user", "content": "你好"}]
  }'
            

# Python示例
import requests

response = requests.post(
    "https://aiagent.crazykangaroo.space/v1/chat/completions",
    headers={
        "Authorization": "Bearer <您的API密钥>",
        "Content-Type": "application/json"
    },
    json={
        "model": "claude-2.1",
        "messages": [{"role": "user", "content": "讲个笑话"}]
    }
)
print(response.json())

用量监控与统计

  • 实时查看「数据看板」：展示按时间/按API密钥的消耗趋势
  • 设置用量预警：达到设定阈值时邮件通知
  • 数据导出：支持CSV格式的历史记录
